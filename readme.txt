Плагин WC SDEK Shipping Calculator добавляет метод доствки для курьерской службы СДЕК

== Description ==

Плагин добалвяет в вашу корзину способы доставки службой СДЕК и рассчитывает их стоимость. Задача плагина именно рассчитать цену за тариф. Он не формирует автоматиечки заявку в курьерской службе. Заявки вы должны составлять самостоятельно в личном кбаинете на сайте cdek.ru

== Installation ==
1. Скачайте .ZIP архив на ваш компьютер.
2. Зайдите в панель управления вашего сайта, в разделе плагины, выберите пунтк "Добавить плагин".
3. Укажите путь к скаченному ранее .ZIP архиву
4. Загрузите архив на сервер.
5. Активируйте плагин.
